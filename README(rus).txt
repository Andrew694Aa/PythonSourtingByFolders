      BELGEN - SortingByFolders Автоматическая сортировка

===========================================================

Описание:
**SortingByFolders** — это инструмент для автоматической сортировки файлов по расширениям, определённым в `config.cfg`.

===========================================================

Возможности:
- Быстрая сортировка всех файлов в папке по категориям.
- Простое редактирование `config.cfg` прямо из интерфейса.
- Логирование ошибок в `sorter_errors.log`.

===========================================================

Как запустить:

1. Перейдите в папку `dist/`.
2. Запустите `main.exe` вручную или используйте `SourtingByFolders.bat`.
3. Интерфейс откроется автоматически.

===========================================================

Советы:
- Расширения и папки задаются в `config.cfg`.  
  Пример:- Если `config.cfg` не найден — он будет создан автоматически.
- Иконка `Sorting_by_folders_Icon.ico` должна лежать рядом с `main.exe`.
