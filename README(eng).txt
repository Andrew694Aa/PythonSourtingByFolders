      BELGEN ‚Äî SortingByFolders üìÅ Automatic File Organizer

===========================================================

üîß Description:
**SortingByFolders** is a tool for automatically sorting files by their extensions as defined in the `config.cfg`.

===========================================================

üìÅ Features:
- Quickly sort all files in a folder by categories.
- Easy editing of `config.cfg` directly from the interface.
- Error logging to `sorter_errors.log`.

===========================================================

üöÄ How to Run:

1. Navigate to the `dist/` folder.
2. Launch `main.exe` manually or run `SortingByFolders.bat`.
3. The user interface will open automatically.

===========================================================

üí° Tips:
- File extensions and destination folders are set in `config.cfg`.  
  Example:- If `config.cfg` is not found, it will be generated automatically.
- The icon file `Sorting_by_folders_Icon.ico` should be placed next to `main.exe`.
